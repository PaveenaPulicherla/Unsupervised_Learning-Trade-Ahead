# Unsupervised_Learning-Trade-Ahead



### Project Overview
This project was developed for Trade&Ahead, a financial consultancy firm seeking to enhance stock portfolio strategies through unsupervised learning. By clustering stocks based on financial indicators, the goal is to identify groups with similar characteristics and minimal correlationâ€”helping investors diversify and mitigate risk.

### Objective
- Group NYSE-listed stocks using unsupervised learning techniques
- Discover patterns across financial metrics to inform investment decisions
- Profile clusters to guide personalized portfolio recommendations

### Dataset Summary
- **Total records**: 340 companies
- **Features**: 15 financial and categorical attributes including:
  - Sector, Sub-industry, Current Price, Price Change, Volatility
  - ROE, Cash Ratio, Net Income, Net Cash Flow, EPS
  - P/E Ratio, P/B Ratio, Estimated Shares Outstanding
